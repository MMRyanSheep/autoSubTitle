# Auto subtitle tanslator / 字幕翻译
本项目基于argostranslate和opencc创建
## (0.1.0 - 更新) nogui版本
### 使用说明
运行main.exe  
分别输入  
请输入翻译语言代码（例如：zh-CT）：zh-CT  
请输入翻译语言代码（例如：zh-CN）：zh-CN  
**目前暂不支持其他语言**

## (0.1.1 - 更新) gui更新
### 使用说明
运行 install.bat  
运行 gui.py  
### 版本说明
**此版本仅在python 3.12.0 环境测试正常**  
目前只支持zh-CT 转 zh-CN 其他语言正在测试中 预计0.1.3版本上线
